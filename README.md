# Ford-Fulkerson Max Flow Visualizer

Interactive animated presentation teaching the Ford-Fulkerson Maximum Flow algorithm (CLRS Chapter 26).

20 slides covering three core concepts: Residual Network, Augmenting Path, and Max-Flow Min-Cut Theorem.

## Quick Start

```bash
pnpm install
pnpm dev
```

## Controls

- **Arrow keys / Space**: Next step
- **Backspace**: Previous step
- **L**: Toggle English / Chinese
- **Ctrl+1-9**: Jump to slide

## Tech Stack

React + TypeScript + Vite + Tailwind CSS + Framer Motion + KaTeX + Zustand
